# Word List

`words.txt` contains 15,921 five-letter English words, extracted from
[dwyl/english-words](https://github.com/dwyl/english-words) (370k+ words,
Unlicense / public domain).

The original list derives from the Moby Project word lists. It was filtered
to lowercase five-letter alphabetic words, deduplicated, and sorted.
